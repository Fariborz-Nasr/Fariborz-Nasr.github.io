Project 1: The New York Social Graph

The project is focused on analyzing the social graph of New York’s social elite based on photo captions from the New York Social Diary website. The goal is to identify the most popular socialites, influential individuals, and tightly coupled pairs within the social network.

To achieve this, the project involves several steps:

  • Getting a list of parties: The project utilizes an API to retrieve a list of parties and their dates from the New York Social Diary website. The API allows for pagination to retrieve all parties, and the data is filtered to include only parties dated on or before December 1, 2014.

  • Histogram of party pages: The number of party pages for each month-year pair is calculated to create a histogram. This provides insights into the frequency and trends of social events over time.

  • Parsing captions: The project focuses on one party initially and retrieves the captions for that party using the API. The captions are then parsed to extract the names of individuals mentioned. Various heuristics and techniques, such as part-of-speech tagging and entity recognition, can be used to identify names in the captions.

  • Creating the social graph: The extracted names from the captions are used to build connections between individuals, forming the social graph. The graph represents the relationships between individuals based on their appearances together in the photos.

The project includes additional considerations, such as handling HTTP request failures, dealing with formatting issues in captions, filtering out unnecessary information, and addressing special cases like couples’ names.

Finally, the project aims to analyze the social graph to find the most connected individuals, influential people, and tightly coupled pairs within the social elite network.
